# 采购订单执行明细表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/purchase-order-detail`
- **Content-Type**: `application/json`
- **表单ID**: `PUR_PurchaseOrderDetailRpt`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": "true",
  "FilterString": [],
  "Model": {
    "FPurchaseOrgIdList": "",
    "FOrderStartDate": "1900-01-01",
    "FOrderEndDate": "1900-01-01",
    "FBeginSupplierId": {
      "FNumber": ""
    },
    "FEndSupplierId": {
      "FNumber": ""
    },
    "FBeginBillNumber": "",
    "FEndBillNumber": "",
    "FBeginMaterialId": {
      "FNumber": ""
    },
    "FEndMaterialId": {
      "FNumber": ""
    },
    "FBeginPurchaser": {
      "FNumber": ""
    },
    "FEndFPurchaser": {
      "FNumber": ""
    },
    "FBusinessType": "",
    "FDocumentStatus": "",
    "FLineStatus": ""
  }
}
```

### 2.2 参数详细说明

||| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
|| --- | ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
||| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FBILLNO,FDate,FSUPPLIERID" |
||| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
||| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
||| Limit                   | int     |  否  | 最大行数，不能超过10000                    | 2000                      |
||| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
||| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
||| FPurchaseOrgIdList      | String  |  是  | 采购组织                                   | ""                        |
||| FOrderStartDate         | String  |  是  | 订单日期（起始）                           | "1900-01-01"              |
||| FOrderEndDate           | String  |  是  | 订单日期（结束）                           | "1900-01-01"              |
||| FBeginSupplierId        | Object  |  否  | 供应商编码（起始）                         | {"FNumber":""}            |
||| FEndSupplierId          | Object  |  否  | 供应商编码（结束）                         | {"FNumber":""}            |
||| FBeginBillNumber        | String  |  否  | 订单编号（起始）                           | ""                        |
||| FEndBillNumber          | String  |  否  | 订单编号（结束）                           | ""                        |
||| FBeginMaterialId        | Object  |  否  | 物料编码（起始）                           | {"FNumber":""}            |
||| FEndMaterialId          | Object  |  否  | 物料编码（结束）                           | {"FNumber":""}            |
||| FBeginPurchaser         | Object  |  否  | 采购员编码（起始）                         | {"FNumber":""}            |
||| FEndFPurchaser          | Object  |  否  | 采购员编码（结束）                         | {"FNumber":""}            |
||| FBusinessType           | String  |  是  | 业务类型                                   | ""                        |
||| FDocumentStatus         | String  |  是  | 单据状态                                   | ""                        |
||| FLineStatus             | String  |  是  | 行状态                                     | ""                        |

||| 8    | 参数错误              |

||| 10   | 未找到对应数据        |
