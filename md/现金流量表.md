# 现金流量表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/cash-flow-report`
- **Content-Type**: `application/json`
- **表单ID**: `GL_Rpt_CashFlow`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": "true",
  "FilterString": [],
  "Model": {
    "FACCTBOOKID": {
      "FNumber": ""
    },
    "FCURRENCYID": "",
    "FSumLevel": "",
    "FByPeriod": "",
    "FByDate": "",
    "FSTARTYEAR": "",
    "FSTARTPERIOD": "",
    "FENDYEAR": "",
    "FENDPERIOD": "",
    "FSTARTDATE": "1900-01-01",
    "FENDDATE": "1900-01-01",
    "FIncNotPost": "false",
    "FShowRateDetail": "false",
    "FCBFORBIDITEM": "false",
    "FFindType": ""
  }
}
```

### 2.2 参数详细说明

||| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
|| --- | ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
||| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FItemTypeDesc,FCashItem,FRowNumber,FAmount,FPercent" |
||| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
||| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
||| Limit                   | int     |  否  | 最大行数，不能超过10000                   | 2000                      |
||| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
||| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
||| FACCTBOOKID            | Object  |  是  | 账簿                                       | {"FNumber":""}            |
||| FCURRENCYID            | String  |  是  | 币别                                       | ""                        |
||| FFindType              | String  |  否  | 单选按钮组                                 | ""                        |
||| FByPeriod              | String  |  否  | 按期间查询                                 | ""                        |
||| FByDate                | String  |  否  | 按日期查询                                 | ""                        |
||| FSumLevel              | String  |  否  | 项目汇总级次                               | ""                        |
||| FSTARTYEAR              | String  |  否  | 会计年度（起始）                           | ""                        |
||| FSTARTPERIOD            | String  |  否  | 会计期间（起始）                           | ""                        |
||| FENDYEAR                | String  |  否  | 会计年度（结束）                           | ""                        |
||| FENDPERIOD              | String  |  否  | 会计期间（结束）                           | ""                        |
||| FSTARTDATE             | String  |  否  | 开始日期                                   | "1900-01-01"              |
||| FENDDATE               | String  |  否  | 结束日期                                   | "1900-01-01"              |
||| FIncNotPost            | String  |  否  | 包括未过账凭证                             | "false"                   |
||| FShowRateDetail        | String  |  否  | 显示汇率变动产生的现金流量明细项目         | "false"                   |
||| FCBFORBIDITEM          | String  |  否  | 不显示禁用项目                             | "false"                   |

## 三、响应字段说明

||| 字段Key                   | 说明                       |
|| --- | ------------------------ | -------------------------- |
||| FItemTypeDesc            | 项目类别                   |
||| FCashItem                | 现金流量项目               |
||| FRowNumber               | 行次                       |
||| FAmount                  | 金额                       |
||| FPercent                 | 比重                       |
