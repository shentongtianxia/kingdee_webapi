# 产品维度利润分析表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/product-profit-analysis-report`
- **Content-Type**: `application/json`
- **表单ID**: `CB_PROSALEPROFITRPT`

### 2.2 参数详细说明

| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
| ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FPRODUCTID,FPRODUCTNAME,FSALEAMOUNT,FCOSTAMOUNT,FSALESPROFIT,FPROFITRATE" |
| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
| Limit                   | int     |  否  | 最大行数，不能超过10000                   | 2000                      |
| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
| FACCTGSYSTEMID         | Object  |  是  | 核算体系编码                               | {"FNumber":""}            |
| FACCTGORGID            | Object  |  是  | 核算组织编码                               | {"FNumber":""}            |
| FACCTPOLICYID          | Object  |  是  | 会计政策编码                               | {"FNumber":""}            |
| FYear                   | String  |  是  | 开始会计年度                               | "2025"                    |
| FPeriod                 | String  |  是  | 开始会计期间                               | "1"                       |
| FEndYear                | String  |  是  | 结束会计年度                               | "2025"                    |
| FEndPeriod              | String  |  是  | 结束会计期间                               | "12"                      |
| FSTARTMATERIALID        | Object  |  否  | 产品编码（起始）                           | {"FNumber":""}            |
| FENDMATERIALID          | Object  |  否  | 产品编码（结束）                           | {"FNumber":""}            |
| FCurrencyID             | Object  |  否  | 币别                                       | {"FNumber":""}            |
| FSTARTBILLNO           | String  |  否  | 单据编号（起始）                           | ""                        |
| FENDBILLNO             | String  |  否  | 单据编号（结束）                           | ""                        |
| FFPROORDERTYPE         | String  |  否  | 业务类型                                   | ""                        |
| FCHXTotal               | String  |  否  | 只显示汇总行                               | "false"                   |
| FSTARTSALEDEPT         | Object  |  否  | 销售部门（起始）                           | {"FNUMBER":""}            |
| FENDSALEDEPT           | Object  |  否  | 销售部门（结束）                           | {"FNUMBER":""}            |
| FSTARTSALER            | Object  |  否  | 销售员（起始）                             | {"FNUMBER":""}            |
| FENDSALER              | Object  |  否  | 销售员（结束）                             | {"FNUMBER":""}            |
| FSTARTCUSTOMER         | Object  |  否  | 客户（起始）                               | {"FNUMBER":""}            |
| FENDCUSTOMER           | Object  |  否  | 客户（结束）                               | {"FNUMBER":""}            |
| FSTARTEXPENSEID        | Object  |  否  | 费用项目（起始）                           | {"FNumber":""}            |
| FENDEXPENSEID          | Object  |  否  | 费用项目（结束）                           | {"FNumber":""}            |
| FSTARTCOSTCID          | Object  |  否  | 成本中心（起始）                           | {"FNumber":""}            |
| FENDCOSTCID            | Object  |  否  | 成本中心（结束）                           | {"FNumber":""}            |
| FISSHOWDETAIL          | String  |  否  | 显示费用项目明细                           | "false"                   |
| FMULBILLTYPE           | String  |  否  | 多选单据类型                               | ""                        |
| FBILLTYPE              | String  |  否  | 单据类型                                   | ""                        |
| FSTARTDATE             | String  |  否  | 开始日期                                   | "1900-01-01"              |
| FENDDATE               | String  |  否  | 结束日期                                   | "1900-01-01"              |
| FSTARTPROORDERID       | String  |  否  | 工单编号（起始）                           | ""                        |
| FENDPROORDERID         | String  |  否  | 工单编号（结束）                           | ""                        |
| FGroupBy               | String  |  否  | 汇总依据                                   | ""                        |
| FAcrossOrgOrder        | String  |  否  | 获取跨组织结算数据                         | "false"                   |
| FCUSTOMERTYPE          | String  |  否  | 客户类型                                   | ""                        |
| FGETCBFROMAR           | String  |  否  | 按应收单获取收入成本                       | "false"                   |
| FISARAPINFO            | String  |  否  | 统计暂估应收单和暂估冲回单                 | "false"                   |
| FRADIOGROUP            | String  |  否  | 单选按钮组                                 | ""                        |
| FRdPeriod              | String  |  否  | 按期间查询                                 | ""                        |
| FRdData                | String  |  否  | 按日期查询                                 | ""                        |
| FIsDisplayPeriod       | String  |  否  | 显示各期间明细                             | "false"                   |
| FSHOWALLAMOUNTFOR      | String  |  否  | 收入按价税合计显示                         | "false"                   |
