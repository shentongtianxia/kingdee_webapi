# 科目余额表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/account-balance`
- **Content-Type**: `application/json`
- **表单ID**: `GL_RPT_AccountBalance`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": true,
  "FilterString": [],
  "Model": {}
}
```

### 2.2 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." |
| SchemeId | String | 否 | 过滤方案内码 |
| StartRow | Int | 否 | 开始行索引，默认0 |
| Limit | Int | 否 | 最大行数，不能超过10000，默认2000 |
| IsVerifyBaseDataField | Boolean | 否 | 是否验证所有的基础资料有效性，默认true |
| FilterString | Array | 否 | 过滤条件 |
| Model | Object | 是 | 表单数据包 |

### 2.3 Model参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| FACCTBOOKID | Object | 账簿（必填），格式：{"FNumber":"xxx"} |
| FCURRENCY | String | 币别（必填） |
| FSTARTYEAR | String | 开始会计年度 |
| FSTARTPERIOD | String | 开始会计期间 |
| FENDYEAR | String | 结束会计年度 |
| FENDPERIOD | String | 结束会计期间 |
| FBALANCELEVEL | String | 科目级别 |
| FSTARTBALANCE | Object | 科目编码（开始），格式：{"FNumber":"xxx"} |
| FENDBALANCE | Object | 科目编码（结束），格式：{"FNumber":"xxx"} |
| FSHOWDETAIL | Boolean | 显示核算维度明细 |
| FFORBIDBALANCE | Boolean | 显示禁用科目 |
| FNOTPOSTVOUCHER | Boolean | 包括未过账凭证 |
| FDEBITORCREDIT | Boolean | 余额按借方、贷方分别小计 |
| FBALANCEZERO | Boolean | 包括余额为零的科目 |
| FNOBUSINESS | Boolean | 包括没有业务发生的科目 |
| FPERIODNOBALANCE | Boolean | 包括本期没有发生额的科目 |
| FYEARNOBALANCE | Boolean | 包括本年没有发生额的科目 |
| FSHOWFULLNAME | Boolean | 显示科目全名 |
| FDETAILSHOWACCT | Boolean | 核算维度明细行显示科目信息 |
| FSHOWDETAILONLY | Boolean | 只显示明细科目 |
| FEXCLUDEADJUSTVCH | Boolean | 不包含调整期凭证 |
| FFLEXDEBITORCREDIT | Boolean | 核算维度余额按借方、贷方分别小计 |
| FSHOWFLEXBYCOL | Boolean | 核算维度分列显示 |

## 三、返回结果

```json
{
  "Result": {
    "IsSuccess": true,
    "RowCount": 0,
    "Rows": []
  }
}
```
