# 存货收发存汇总表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/inventory-in-out-summary-report`
- **Content-Type**: `application/json`
- **表单ID**: `HS_INOUTSTOCKSUMMARYRPT`

### 2.2 参数详细说明

| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
| ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FMATERIALBASEID,FMATERIALNAME,FMODEL,FINITQty,FINITAMOUNT,FRECEIVEQty,FRECEIVEAmount,FSENDQty,FSENDAmount,FENDQty,FENDAMOUNT" |
| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
| Limit                   | int     |  否  | 最大行数，不能超过10000                   | 2000                      |
| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
| FACCTGSYSTEMID         | Object  |  是  | 核算体系编码                               | {"FNumber":""}            |
| FACCTGORGID            | Object  |  是  | 核算组织编码                               | {"FNumber":""}            |
| FACCTPOLICYID          | Object  |  是  | 会计政策编码                               | {"FNumber":""}            |
| FYear                   | String  |  否  | 开始会计年度                               | "2025"                    |
| FPeriod                 | String  |  否  | 开始会计期间                               | "1"                       |
| FENDYEAR                | String  |  否  | 结束会计年度                               | "2025"                    |
| FEndPeriod              | String  |  否  | 结束会计期间                               | "12"                      |
| FMATERIALID             | Object  |  否  | 物料编码（起始）                           | {"FNumber":""}            |
| FENDMATERIALID          | Object  |  否  | 物料编码（结束）                           | {"FNumber":""}            |
| FEXPENID               | Object  |  否  | 费用项目（起始）                           | {"FNumber":""}            |
| FENDEXPENID            | Object  |  否  | 费用项目（结束）                           | {"FNumber":""}            |
| FACCTGRANGEID          | Object  |  否  | 核算范围                                   | {"FNumber":""}            |
| FOwnerID               | Object  |  否  | 货主                                       | {"FNumber":""}            |
| FSTOCKORGID            | Object  |  否  | 库存组织                                   | {"FNumber":""}            |
| FSTOCKSTATUSID         | Array   |  否  | 库存状态                                   | [{"FNumber":""}]          |
| FSTOCKId               | Object  |  否  | 仓库编码（起始）                           | {"FNumber":""}            |
| FENDSTOCKID            | Object  |  否  | 仓库编码（结束）                           | {"FNumber":""}            |
| FMATERTYPEID           | Object  |  否  | 存货类别                                   | {"FNUMBER":""}            |
| FCOMBOTotalType        | String  |  是  | 汇总依据                                   | ""                        |
| FDimType               | String  |  是  | 显示维度                                   | ""                        |
| FCHXEXPENSE            | String  |  否  | 显示费用项目明细                           | "false"                   |
| FCHXTotal              | String  |  否  | 只显示汇总行                               | "false"                   |
| FCHXNOINOUT            | String  |  否  | 无收发不显示                               | "false"                   |
| FCHXNOCOSTALLOT        | String  |  否  | 不统计核算组织内调拨单据                   | "false"                   |
| FIsDisplayPeriod       | String  |  否  | 显示各期间明细                             | "false"                   |
| FCHXNOSTOCKADJ         | String  |  否  | 不统计库存调整单据                         | "false"                   |
| FCOMBOSTATUS           | String  |  否  | 单据状态                                   | ""                        |
| FCurrencyID            | Object  |  否  | 币别                                       | {"FNumber":""}            |

| 7    | 许可错误              |
| 8    | 参数错误              |
| 9    | 指定字段/值不存在    |
| 10   | 未找到对应数据        |
