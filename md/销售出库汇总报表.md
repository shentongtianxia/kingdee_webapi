# 销售出库汇总报表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/sales-outbound-summary`
- **Content-Type**: `application/json`
- **表单ID**: `SAL_OutStockSummaryRpt`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": true,
  "FilterString": [],
  "Model": {
    "FStartDate": "1900-01-01",
    "FEndDate": "1900-01-01",
    "FStartMaterial": {
      "FNUMBER": ""
    },
    "FEndMaterial": {
      "FNUMBER": ""
    },
    "FStartCustomer": {
      "FNumber": ""
    },
    "FEndCustomer": {
      "FNumber": ""
    },
    "FStartDepartment": {
      "FNUMBER": ""
    },
    "FEndDepartment": {
      "FNUMBER": ""
    },
    "FBillStatus": "",
    "FIsIncludeInnerExchangeBill": false,
    "FIsIncludeGift": false,
    "FIsIncludeSerMat": false,
    "FSaleOrgId": "",
    "FSettleOrgId": "",
    "FSuite": "",
    "FIsGroupCust": false,
    "FOnlyOutReturnData": false,
    "FIncludeRefundNoGoods": false
  }
}
```

### 2.2 参数详细说明

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|:----:|------|--------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." | "FDEPTNAME,FSALERNAME" |
| SchemeId | String | 否 | 过滤方案内码 | "" |
| StartRow | int | 否 | 开始行索引 | 0 |
| Limit | int | 否 | 最大行数，不能超过10000 | 2000 |
| IsVerifyBaseDataField | boolean | 否 | 是否验证所有的基础资料有效性 | true |
| FilterString | Array | 否 | 过滤条件 | [] |
| FStartDate | String | 是 | 起始日期（格式：yyyy-MM-dd） | "2025-01-01" |
| FEndDate | String | 是 | 结束日期（格式：yyyy-MM-dd） | "2025-01-31" |
| FStartMaterial | Object | 否 | 起始物料 | {"FNUMBER":"M001"} |
| FEndMaterial | Object | 否 | 结束物料 | {"FNUMBER":"M999"} |
| FStartCustomer | Object | 否 | 起始客户 | {"FNumber":"C001"} |
| FEndCustomer | Object | 否 | 结束客户 | {"FNumber":"C999"} |
| FStartDepartment | Object | 否 | 起始销售部门 | {"FNUMBER":"D001"} |
| FEndDepartment | Object | 否 | 结束销售部门 | {"FNUMBER":"D999"} |
| FBillStatus | String | 否 | 单据状态 | "" |
| FSaleOrgId | String | 否 | 销售组织 | "" |
| FSettleOrgId | String | 否 | 结算组织 | "" |
| FSuite | String | 否 | 统计套件 | "" |
| FIsIncludeInnerExchangeBill | boolean | 否 | 包含跨组织结算单据 | false |
| FIsIncludeGift | boolean | 否 | 包含赠品 | false |
| FIsIncludeSerMat | boolean | 否 | 包含服务类物料 | false |
| FIsGroupCust | boolean | 否 | 集团客户 | false |
| FOnlyOutReturnData | boolean | 否 | 仅取出库退货单数据分析 | false |
| FIncludeRefundNoGoods | boolean | 否 | 包含仅退款不退货 | false |
