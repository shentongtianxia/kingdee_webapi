# 生产入库月报表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/production-inbound-monthly`
- **Content-Type**: `application/json`
- **表单ID**: `PRD_InStockMonthRpt`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": true,
  "FilterString": [],
  "Model": {
    "FPickMtrlMonth": "1900-01-01",
    "FWorkShopId": [
      {
        "FNumber": ""
      }
    ],
    "FProdNumId": [
      {
        "FNumber": ""
      }
    ],
    "FSumBasis": "",
    "FProdTypeId": [
      {
        "FNumber": ""
      }
    ],
    "FPrdOrgId": "",
    "FApprBeginDate": "1900-01-01",
    "FApprEndDate": "1900-01-01",
    "FMonDateChk": false,
    "FRangeDateChk": false,
    "FMoBillTypeId": [
      {
        "FNumber": ""
      }
    ],
    "FSumUnit": "",
    "FAccSource": "",
    "FDocStatus": ""
  }
}
```

### 2.2 参数详细说明

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|:----:|------|--------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." | "FWorkShopId,FProdName" |
| SchemeId | String | 否 | 过滤方案内码 | "" |
| StartRow | int | 否 | 开始行索引 | 0 |
| Limit | int | 否 | 最大行数，不能超过10000 | 2000 |
| IsVerifyBaseDataField | boolean | 否 | 是否验证所有的基础资料有效性 | true |
| FilterString | Array | 否 | 过滤条件 | [] |
| FPickMtrlMonth | String | 否 | 月份（格式：yyyy-MM-dd） | "2025-01-01" |
| FWorkShopId | Array | 否 | 生产车间 | [{"FNumber":"WC001"}] |
| FProdNumId | Array | 否 | 产品编码 | [{"FNumber":"P001"}] |
| FSumBasis | String | 是 | 汇总依据 | "" |
| FProdTypeId | Array | 否 | 物料组 | [{"FNumber":"T001"}] |
| FPrdOrgId | String | 是 | 生产组织 | "" |
| FApprBeginDate | String | 否 | 期间从（格式：yyyy-MM-dd） | "2025-01-01" |
| FApprEndDate | String | 否 | 至（格式：yyyy-MM-dd） | "2025-01-31" |
| FMonDateChk | boolean | 否 | 月份期间选择 | false |
| FRangeDateChk | boolean | 否 | 日期范围选择 | false |
| FMoBillTypeId | Array | 否 | 单据类型 | [{"FNumber":""}] |
| FSumUnit | String | 是 | 统计单位 | "" |
| FAccSource | String | 否 | 取数来源 | "" |
| FDocStatus | String | 是 | 单据状态 | "" |

## 三、响应字段说明

| 字段Key | 说明 |
|---------|------|
| FPickMtrlMonth | 月份/期间 |
| FWorkShopId | 生产车间 |
| FProdTypeId | 物料组 |
| FPrdOrgId | 生产组织 |
| FMATERIALID | 产品编码 |
| FProdName | 产品名称 |
| FSpecModel | 规格型号 |
| FUnitId | 单位 |
| FPASSINSTOCKQTY | 合格品入库数量 |
| FFAILINSTOCKQTY | 不合格品入库数量 |
| FSCRAPINSTOCKQTY | 报废品入库数量 |
| FRETURNINSTOCKQTY | 返工品入库数量 |
| FOUTSTOCKQTY | 退库数量 |
| FACTUALINSTOCKQTY | 实际入库数量 |
| FAUXUNITID | 库存辅单位 |
| FSECPASSINSTOCKQTY | 合格品入库数量（辅） |
| FSECFAILINSTOCKQTY | 不合格品入库数量（辅） |
| FSECSCRAPINSTOCKQTY | 报废品入库数量（辅） |
| FSECRETURNINSTOCKQTY | 返工品入库数量（辅） |
| FSECOUTSTOCKQTY | 退库数量（辅） |
| FSECACTUALINSTOCKQTY | 实际入库数量（辅） |
