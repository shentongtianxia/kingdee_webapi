# 销售订单执行明细表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/sale-order-detail`
- **Content-Type**: `application/json`
- **表单ID**: `SAL_DetailReport`

### 2.2 参数详细说明

|| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
|| ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
|| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FBILLNO,FDate,FCUSTID"   |
|| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
|| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
|| Limit                   | int     |  否  | 最大行数，不能超过10000                    | 2000                      |
|| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
|| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
|| FSaleOrgList            | String  |  是  | 组织                                       | ""                        |
|| FMoneyType              | Object  |  否  | 币别                                       | {"FNumber":""}            |
|| FSoFromDate             | String  |  是  | 订单日期（起始）                           | "1900-01-01"              |
|| FSoToDate               | String  |  是  | 订单日期（结束）                           | "1900-01-01"              |
|| FCustomerFrom           | Object  |  否  | 客户编码（起始）                           | {"FNumber":""}            |
|| FCustomerTo             | Object  |  否  | 客户编码（结束）                           | {"FNumber":""}            |
|| FSalesFrom              | Object  |  否  | 销售员编码（起始）                         | {"FNumber":""}            |
|| FSalesTo                | Object  |  否  | 销售员编码（结束）                         | {"FNumber":""}            |
|| FMaterialFrom           | Object  |  否  | 物料编码（起始）                           | {"FNumber":""}            |
|| FMaterialTo             | Object  |  否  | 物料编码（结束）                           | {"FNumber":""}            |
|| FSaleOFrom              | String  |  否  | 销售订单（起始）                           | ""                        |
|| FSaleOTo                | String  |  否  | 销售订单（结束）                           | ""                        |
|| FDelliveryDateFrom      | String  |  否  | 要货日期（起始）                           | "1900-01-01"              |
|| FDelliveryDateTo        | String  |  否  | 要货日期（结束）                           | "1900-01-01"              |
|| FFormCloseStatus        | String  |  是  | 整单关闭状态                               | ""                        |
|| FFormStatus             | String  |  是  | 单据状态                                   | ""                        |
|| FPriceFrom              | String  |  是  | 单价来源                                   | ""                        |
|| FBusCloseStatus         | String  |  是  | 行业务关闭状态                             | ""                        |
|| FMergingSOHeader        | String  |  否  | 查询时不合并展示订单的表头信息              | "false"                   |
|| FIncludedUnfilledOrders | String  |  否  | 包含未执行的销售订单                       | "false"                   |
|| FIsRecWithMat           | String  |  否  | 按物料明细收款方式查询                     | "false"                   |
|| FCustGroup              | Object  |  否  | 客户分组                                   | {"FNumber":""}            |
|| FSuite                  | String  |  是  | 统计套件                                   | ""                        |
|| FSetAccountType         | String  |  是  | 立账类型                                   | ""                        |
|| FIsGroup                | String  |  否  | 集团客户                                   | "false"                   |
|| FIncludedFree           | String  |  否  | 包含赠品                                   | "false"                   |
