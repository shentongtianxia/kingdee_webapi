# 采购资料计划报表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/purchase-feed-stock`
- **Content-Type**: `application/json`
- **表单ID**: `PUR_FeedStockRpt`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": true,
  "FilterString": [],
  "Model": {
    "FPurchaseOrgIdList": "",
    "FOrderStartDate": "1900-01-01",
    "FOrderEndDate": "1900-01-01",
    "FBeginSupplierId": {
      "FNUMBER": ""
    },
    "FEndSupplierId": {
      "FNUMBER": ""
    },
    "FBeginBillNumber": "",
    "FEndBillNumber": "",
    "FBeginMaterialId": {
      "FNUMBER": ""
    },
    "FEndMaterialId": {
      "FNUMBER": ""
    },
    "FBeginPurchaser": {
      "FNUMBER": ""
    },
    "FEndFPurchaser": {
      "FNUMBER": ""
    },
    "FBusinessType": "",
    "FDocumentStatus": "",
    "FLineStatus": "",
    "FPushStartDate": "1900-01-01",
    "FPushEndDate": "1900-01-01",
    "FBeginLackBillNumber": "",
    "FEndLackBillNumber": ""
  }
}
```

### 2.2 参数详细说明

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|:----:|------|--------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." | "FBillno,FSupplierName,FMaterialName" |
| SchemeId | String | 否 | 过滤方案内码 | "" |
| StartRow | int | 否 | 开始行索引 | 0 |
| Limit | int | 否 | 最大行数，不能超过10000 | 2000 |
| IsVerifyBaseDataField | boolean | 否 | 是否验证所有的基础资料有效性 | true |
| FilterString | Array | 否 | 过滤条件 | [] |
| FPurchaseOrgIdList | String | 是 | 采购组织 | "" |
| FOrderStartDate | String | 是 | 订单日期（格式：yyyy-MM-dd） | "2025-01-01" |
| FOrderEndDate | String | 是 | 结束日期（格式：yyyy-MM-dd） | "2025-01-31" |
| FBeginSupplierId | Object | 否 | 起始供应商编码 | {"FNUMBER":"S001"} |
| FEndSupplierId | Object | 否 | 结束供应商编码 | {"FNUMBER":"S999"} |
| FBeginBillNumber | String | 否 | 订单编号（起始） | "" |
| FEndBillNumber | String | 否 | 订单编号（结束） | "" |
| FBeginMaterialId | Object | 否 | 起始物料编码 | {"FNUMBER":"M001"} |
| FEndMaterialId | Object | 否 | 结束物料编码 | {"FNUMBER":"M999"} |
| FBeginPurchaser | Object | 否 | 起始采购员编码 | {"FNUMBER":"EMP001"} |
| FEndFPurchaser | Object | 否 | 结束采购员编码 | {"FNUMBER":"EMP999"} |
| FBusinessType | String | 是 | 业务类型 | "" |
| FDocumentStatus | String | 是 | 单据状态 | "" |
| FLineStatus | String | 是 | 行状态 | "" |
| FPushStartDate | String | 是 | 推送日期（格式：yyyy-MM-dd） | "2025-01-01" |
| FPushEndDate | String | 是 | 推送结束日期（格式：yyyy-MM-dd） | "2025-01-31" |
| FBeginLackBillNumber | String | 否 | 推送单号（起始） | "" |
| FEndLackBillNumber | String | 否 | 推送单号（结束） | "" |
