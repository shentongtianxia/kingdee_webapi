# 多账簿日报表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/kingdeedataset/multi-daily`
- **Content-Type**: `application/json`
- **表单ID**: `GL_RPT_DailyReport`

## 二、请求参数说明

### 2.2 参数详细说明

| 参数名 | 类型 | 必填 | 说明 | 示例值 |
|--------|------|:----:|------|--------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." | "FDate,FAccountNumber,FAccountName" |
| SchemeId | String | 否 | 过滤方案内码 | "" |
| StartRow | int | 否 | 开始行索引 | 0 |
| Limit | int | 否 | 最大行数，不能超过10000 | 2000 |
| IsVerifyBaseDataField | boolean | 否 | 是否验证所有的基础资料有效性 | true |
| FilterString | Array | 否 | 过滤条件 | [] |
| FStartDate | String | 是 | 起始日期（格式：yyyy-MM-dd） | "2025-01-01" |
| FEndDate | String | 是 | 结束日期（格式：yyyy-MM-dd） | "2025-01-31" |
| FStartBalance | Object | 否 | 起始科目编码 | {"FNumber":"1001"} |
| FEndBalance | Object | 否 | 结束科目编码 | {"FNumber":"1002"} |
| FCurrencyId | String | 否 | 币别 | "" |
| FSchemeID | Object | 否 | 账簿隶属关系 | {"FNUMBER":""} |
| FUseAcctBookGroup | String | 否 | 按账簿查询 | "" |
| FUseAcctBookMergeGroup | String | 否 | 按账簿隶属关系查询 | "" |
| FBalance | String | 否 | 科目编码 | "" |
| FNoConBalance | String | 否 | 非连续科目范围查询 | "" |
| FConBalance | String | 否 | 连续科目范围查询 | "" |
| FStartBalanceLevel | String | 否 | 科目级别（起始） | "" |
| FEndBalanceLevel | String | 否 | 科目级别（结束） | "" |
| FShowAcctItems | boolean | 否 | 显示核算维度明细 | false |
| FNotPostVoucher | boolean | 否 | 包括未过账凭证 | false |
| FDetailBalance | boolean | 否 | 只显示明细项目 | false |
| FNoAmount | boolean | 否 | 无发生额不显示 | false |
| FLevel | String | 否 | 合并层级 | "" |
| FLevelId | String | 否 | 合并层级ID | "" |
| FZeroAndNoBalance | boolean | 否 | 余额为零且无发生额不显示 | false |
| FMultiAcctBookId | Array | 否 | 账簿 | [{"FNumber":""}] |
| FShowAcctOnEveryCol | boolean | 否 | 核算维度明细行显示科目信息 | false |
| FShowZeroBalance | boolean | 否 | 余额为零不显示 | false |
| FSameDirection | boolean | 否 | 余额方向与科目余额方向一致 | false |
| FExcludeAdjustVch | boolean | 否 | 不包含调整期凭证 | false |
| FForbidBalance | boolean | 否 | 显示禁用科目 | false |
| FShowTotal | boolean | 否 | 显示合计 | false |
| FShowFullName | boolean | 否 | 显示科目全名 | false |
| FShowQty | boolean | 否 | 显示数量单价 | false |
| FTotalByDay | boolean | 否 | 按日显示 | false |
| FCalByDefaultUnit | boolean | 否 | 按计量单位组默认单位核算 | false |
| FExcludeProfitVch | boolean | 否 | 不包含结转损益凭证 | false |
| FShowChildDetail | boolean | 否 | 展开下级明细 | false |
| FBalanceRadioGrp | String | 否 | 单选按钮组 | "" |
| FACCTBOOKRadioGrp | String | 否 | 单选按钮组 | "" |
