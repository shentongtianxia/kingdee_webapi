# 委外订单执行明细表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/sub-order-detail`
- **Content-Type**: `application/json`
- **表单ID**: `SUB_ROExecuteDetailRpt`

## 二、请求参数说明

### 2.2 参数详细说明

||| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
||| ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
||| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FBILLNO,FDate,FCUSTID"   |
||| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
||| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
||| Limit                   | int     |  否  | 最大行数，不能超过10000                    | 2000                      |
||| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
||| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
||| FEndRONumber            | String  |  否  | 委外订单编号（结束）                       | ""                        |
||| FBeginRONumber          | String  |  否  | 委外订单编号（起始）                       | ""                        |
||| FCtrlItem               | String  |  是  | 领料计算                                   | ""                        |
||| FPurOrgId               | Object  |  否  | 采购组织                                   | {"FNumber":""}            |
||| FStatus                 | String  |  是  | 业务状态                                   | ""                        |
||| FEndConveyDate          | String  |  是  | 下达日期（结束）                           | "1900-01-01"              |
||| FEndPlanStartDate       | String  |  是  | 计划开工日期（结束）                       | "1900-01-01"              |
||| FEndMaterialId          | Object  |  否  | 产品编码（结束）                           | {"FNumber":""}            |
||| FBeginPlanStartDate     | String  |  是  | 计划开工日期（起始）                       | "1900-01-01"              |
||| FBeginPlanFinishDate    | String  |  是  | 计划完工日期（起始）                       | "1900-01-01"              |
||| FEndPlanFinishDate      | String  |  是  | 计划完工日期（结束）                       | "1900-01-01"              |
||| FBeginMaterialId        | Object  |  否  | 产品编码（起始）                           | {"FNumber":""}            |
||| FBeginConveyDate        | String  |  是  | 下达日期（起始）                           | "1900-01-01"              |
||| FROBillType             | String  |  是  | 单据类型                                   | ""                        |
||| FSubOrgId               | String  |  否  | 委外组织                                   | ""                        |
||| FSuppliers              | Array   |  否  | 供应商                                     | [{"FNumber":""}]          |
||| FConveyDateInvolve      | String  |  否  | 下达日期参与查询                           | "false"                   |
||| FStartDateInvolve       | String  |  否  | 计划开工日期参与查询                       | "false"                   |
||| FFinishDateInvolve      | String  |  否  | 计划完工日期参与查询                       | "false"                   |
||| FMulMaterial            | Array   |  否  | 产品编码                                   | [{"FNumber":""}]          |
