# 现金日报表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/cash-daily`
- **Content-Type**: `application/json`
- **表单ID**: `CN_CashDailyReport`

## 二、请求参数说明

### 2.1 请求结构

```json
{
  "FieldKeys": "",
  "SchemeId": "",
  "StartRow": 0,
  "Limit": 2000,
  "IsVerifyBaseDataField": true,
  "FilterString": [],
  "Model": {}
}
```

### 2.2 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| FieldKeys | String | 是 | 需查询的字段key集合，格式："key1,key2,..." |
| SchemeId | String | 否 | 过滤方案内码 |
| StartRow | Int | 否 | 开始行索引，默认0 |
| Limit | Int | 否 | 最大行数，不能超过10000，默认2000 |
| IsVerifyBaseDataField | Boolean | 否 | 是否验证所有的基础资料有效性，默认true |
| FilterString | Array | 否 | 过滤条件 |
| Model | Object | 是 | 表单数据包 |

### 2.3 Model参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| FOrgID | Array | 组织，格式：[{"FNumber":"101"}] |
| FCurrencyIds | String | 币别 |
| FStartDate | String | 开始日期，格式："yyyy-MM-dd" |
| FEndDate | String | 结束日期，格式："yyyy-MM-dd" |
| FNotAudit | Boolean | 未审核 |
| FMyCurrency | 我方币别 |
| FMyPayOrg | Boolean | 我方收付组织 |
| FAffiliation | Object | 归属，格式：{"FNUMBER":"xxx"} |
| FAllAcct | Boolean | 全部账户 |
| FMyCurrencySum | Boolean | 我方币别合计 |
| FCASHACCTID | Array | 现金科目，格式：[{"FNumber":"xxx"}] |
| FCurrencySubTotal | Boolean | 币别小计 |

## 三、返回结果

```json
{
  "Result": {
    "IsSuccess": true,
    "RowCount": 0,
    "Rows": []
  }
}
```
