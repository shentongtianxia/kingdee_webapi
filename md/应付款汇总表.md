# 应付款汇总表接口文档

## 一、接口说明

- **接口地址**: `POST /api/k3cloud/reports/ap-summary-report`
- **Content-Type**: `application/json`
- **表单ID**: `AP_SumReport`

### 2.2 参数详细说明

||| 参数名                  | 类型    | 必填 | 说明                                       | 示例值                    |
|| --- | ----------------------- | ------- | :--: | ------------------------------------------ | ------------------------- |
||| FieldKeys               | String  |  是  | 需查询的字段key集合，格式："key1,key2,..." | "FCONTACTUNITNUMBER,FCONTACTUNITNAME,FAMOUNT,FREALAMOUNT" |
||| SchemeId                | String  |  否  | 过滤方案内码                               | ""                        |
||| StartRow                | int     |  否  | 开始行索引                                 | 0                         |
||| Limit                   | int     |  否  | 最大行数，不能超过10000                    | 2000                      |
||| IsVerifyBaseDataField   | boolean |  否  | 是否验证所有的基础资料有效性               | true                      |
||| FilterString            | Array   |  否  | 过滤条件                                   | []                        |
||| FSettleOrgLst           | String  |  是  | 结算组织                                   | ""                        |
||| FCONTACTUNITTYPE       | String  |  否  | 往来单位类型                               | ""                        |
||| FCONTACTUNITFrom       | Object  |  否  | 往来单位（起始）                           | {"FNumber":""}            |
||| FCONTACTUNITTo         | Object  |  否  | 往来单位（结束）                           | {"FNumber":""}            |
||| FBeginDate              | String  |  是  | 开始日期                                   | "1900-01-01"              |
||| FEndDate                | String  |  是  | 结束日期                                   | "1900-01-01"              |
||| FNOAUDIT                | String  |  否  | 包括未审核单据                             | "false"                   |
||| FNoShowForNoOccur       | String  |  否  | 无发生额不显示                             | "false"                   |
||| FNoShowForNoLeft        | String  |  否  | 余额为零不显示                             | "false"                   |
||| FNoShowForBoth          | String  |  否  | 无发生额且余额为零不显示                   | "false"                   |
||| FOutSettle              | String  |  否  | 外部结算                                   | "false"                   |
||| FInSettle               | String  |  否  | 内部结算                                   | "false"                   |
||| FAccountSystem          | Object  |  否  | 会计核算体系                               | {"FNumber":""}            |
||| FAffiliation           | Object  |  否  | 隶属方案                                   | {"FNAME":""}              |
||| FCurrencyFrom           | Array   |  否  | 币别                                       | [{"FNumber":""}]          |
||| FDEFAULTACCTCALENDARID  | int     |  否  | 默认会计日历                               | 0                         |
||| FDateRadioGrp           | String  |  否  | 单选按钮组                                 | ""                        |
||| FUSEDATE               | String  |  否  | 按业务日期查询                             | ""                        |
||| FUSEPERIOD              | String  |  否  | 按期间查询                                 | ""                        |
||| FOnlyShowPayEvaluate    | String  |  否  | 仅显示已入库未应付                         | "false"                   |
||| FIncludePayEvaluate     | String  |  否  | 包含已入库未应付                           | "false"                   |
||| FSTARTPERIOD            | String  |  否  | 会计期间（起始）                           | ""                        |
||| FSTARTYEAR              | String  |  否  | 会计年度（起始）                           | ""                        |
||| FIncludePayEvaluate_New | String  |  否  | 包含暂估应付                               | "false"                   |
||| FOnlyShowPayEvaluate_New| String  |  否  | 仅显示暂估应付                             | "false"                   |
||| FENDYEAR                | String  |  否  | 会计年度（结束）                           | ""                        |
||| FENDPERIOD              | String  |  否  | 会计期间（结束）                           | ""                        |
||| FShowLocal              | String  |  否  | 显示本位币                                 | "false"                   |
||| FShowSumLocal           | String  |  否  | 显示综合本位币小计                         | "false"                   |
||| FNoPrePayment           | String  |  否  | 不包含预付款                               | "false"                   |
||| FOnlyShowPrePayment     | String  |  否  | 仅显示预付款                               | "false"                   |
||| FShowAmountInCost       | String  |  否  | 仅包含计入成本金额                         | "false"                   |
||| FMULCONTACT             | String  |  否  | 多选                                       | ""                        |
||| FGROUPSUPPLIER          | String  |  否  | 集团供应商                                 | "false"                   |
||| FCONTACTUNITMUL         | String  |  否  | 往来单位                                   | ""                        |
||| FPRESETBASE1            | Array   |  否  | 预设基础资料1                              | [{"FNumber":""}]          |
||| FPRESETBASE2            | Array   |  否  | 预设基础资料2                              | [{"FNumber":""}]          |
||| FPERIODAMOUNT           | int     |  否  | 期末余额                                   | 0                         |
||| FTOPERIODAMOUNT         | int     |  否  | 至期末余额                                 | 0                         |
||| FCheckPeriod            | String  |  否  | 复选框                                     | "false"                   |
||| FShowMatchBill          | String  |  否  | 显示核销单                                 | "false"                   |

||| 7    | 许可错误              |
||| 8    | 参数错误              |

||| 14   | 调用限制              |
